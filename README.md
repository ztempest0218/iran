# iran
Visualizing Iranian Attacks on the UAE

1. The raw data consists of daily image reports published by the UAE Ministry of Defense on X during the wartime period, and the files are stored in the `raw` folder.

2. Since the data was originally unstructured, the process initially relied on manual input. However, by utilizing ChatGPT, the image data was easily transformed into a structured CSV dataset (`iran.csv`).

3. The dataset was then visualized using `iran.R`, which generated the final figure `iran_attacks.png`.


日本語

1. rawデータは、UAE国防省が戦時中にXで毎日公開していた画像データであり、`raw` フォルダに保存している。

2. 非構造化データであるため、ChatGPT を活用することで、画像データを構造化された CSV データ（`iran.csv`）へ整理した。

3. `iran.R` を用いてデータ可視化を行い、最終的に `iran_attacks.png` を生成した。
